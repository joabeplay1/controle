# Omega Factory ⚡

Gerador de software com IA — projeto completo, pronto para rodar localmente.

## Requisitos
- Node.js 18+

## Como usar

```bash
npm install
npm run dev
```

Abra o endereço exibido no terminal (geralmente http://localhost:5173).

## Estrutura do projeto

- `index.html` — página principal
- `src/style.css` — estilos
- `src/script.js` — lógica do aplicativo
- `package.json` / `vite.config.js` — configuração do projeto
- `.gitignore` — arquivos ignorados pelo git

## Build para produção

```bash
npm run build
npm run preview
```

## Uso
1. Cole sua API Key (Google Gemini ou Groq) na barra lateral.
2. Use o Gerador de Prompts ou escreva seu prompt direto.
3. Clique em Gerar Projeto e visualize/baixe o resultado.
